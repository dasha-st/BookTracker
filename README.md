# BookTracker
Решение практического кейса №1: Совместная разработка c Git и GitHub

## Описание
Программа «Трекер прочитанных книг». Предназначена для хранения информации о прочитанных книгах.
Функции:
* Добавить книгу - создать новую запись о прочитанной книге в базе
* Показать все книги - вывести список всех прочитанных книг
  ```log
  -------------------------
  Список прочитанных книг
  -------------------------
  19-05-2026: Толстой Лев - `Война и Мир`. Рейтинг: 4
  19-05-2026: Роулинг Джоан - `Гарри Поттер и Философский камень`. Рейтинг: 5
  19-05-2026: Твен Марк - `Том Сойер`. Рейтинг: 3
  19-05-2026: Роулинг Джоан - `Гарри Поттер и Тайная комната`. Рейтинг: 5
  19-05-2026: Роулинг Джоан - `Гарри Поттер и Узник Азкабана`. Рейтинг: 4
  12-01-1984: Оруэлл Джорж - `1984`. Рейтинг: 5
  10-05-2026: Бунин Иван - `Тёмная аллея`. Рейтинг: 4
  ```
* Показать среднюю оценку - на основании всех записей в базе формирует среднюю оценку
  ```log
  -------------------------
  Средняя оценка по книгам
  -------------------------
  Всего книг прочитано: 7
  Средняя оценка: 4.29
  ```
* Статистика по авторам - по всем найденным в базе авторам выводит количество книг, среднюю оценку и 
список прочитанных книг
  ```log
  -------------------------
  Статистика по авторам
  -------------------------
  Лев Толстой
          Всего прочитано: 1 книг
          Средний рейтинг: 4.0
          Книги:
                  - Война и Мир
  Джоан Роулинг
          Всего прочитано: 3 книг
          Средний рейтинг: 4.67
          Книги:
                  - Гарри Поттер и Философский камень
                  - Гарри Поттер и Тайная комната
                  - Гарри Поттер и Узник Азкабана
  Марк Твен
          Всего прочитано: 1 книг
          Средний рейтинг: 3.0
          Книги:
                  - Том Сойер
  Джорж Оруэлл
          Всего прочитано: 1 книг
          Средний рейтинг: 5.0
          Книги:
                  - 1984
  Иван Бунин
          Всего прочитано: 1 книг
          Средний рейтинг: 4.0
          Книги:
                  - Тёмная аллея
  ```
* Удалить книгу - предлагает пользователю список авторов на выбор, после выбора предлагает список книг
с возможностью указать номер книги для удаления.
  ```log
  -------------------------
  Удаление книги
  -------------------------
  Авторы в базе данных:
          1       Лев Толстой
          2       Джоан Роулинг
          3       Марк Твен
          4       Джорж Оруэлл
          5       Иван Бунин
  Введите номер автора, чью книгу хотите удалить
  1
  Книги выбранного автора `Лев Толстой`:
          1       10-01-2026: Толстой Лев - `Война и Мир`. Рейтинг: 4
  Введите номер книги, которую хотите удалить
  1
  Книга успешно удалена
  ```

## Запуск
```commandline
python main.py
```

## Требования
### Требования к Git и GitHub
* [x] Создать публичный репозиторий на GitHub.
* [x] Первый коммит в ветке main: файлы README.md, .gitignore, пустой books.json (можно []).

### Требования к данным
* Книга:
  * Автор (Имя, Фамилия)
  * Оценка (1 - 5)
  * Дата (Будем принимать в формате ДД-ММ-ГГГГ)

### Разработку вести в отдельных ветках:
* [x] feature/add-book — функционал добавления книги
* [x] feature/list-and-stats — вывод списка и статистика
* [x] feature/delete — удаление книги
* [x] Минимум одна ветка должна вливаться в main через Pull Request на GitHub.
* [x] Создать минимум один Issue и закрыть его коммитом.
* [x] Коммиты осмысленные (можно на русском языке).
* [x] README.md заполнить по окончании работы: что делает приложение, как запустить, пример работы.
