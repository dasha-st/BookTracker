# BookTracker
Решение практического кейса №1: Совместная разработка c Git и GitHub

## Описание
Программа «Трекер прочитанных книг». Предназначена для хранения информации о прочитанных книгах.
Функции:
* Добавить книгу - создать новую запись о прочитанной книге в базе
* Показать все книги - вывести список всех прочитанных книг
* Показать среднюю оценку - на основании всех записей в базе формирует среднюю оценку
* Статистика по авторам - по всем найденным в базе авторам выводит количество книг, среднюю оценку и 
список прочитанных книг
* Удалить книгу - предлагает пользователю список авторов на выбор, после выбора предлагает список книг
с возможностью указать номер книги для удаления.

## Запуск
```commandline
python main.py
```
